O aplikaci

Tato aplikace umožňuje sledovat živé vysílání České televize prostřednictvím streamů. Vyberte si kanál, který chcete sledovat kliknutím na příslušnou dlaždici.

Aplikace používá oficiální API České televize a přehrává obsah pomocí FFmpeg.

Poznámka: Pro použití této aplikace je nutné mít platné poplatky za rozhlasové a televizní vysílání.
